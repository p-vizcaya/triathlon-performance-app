# Ironman application data

Public deployment data for the long-distance triathlon application options.

## User-facing modalities

- `Ironman 70.3`: Media Distancia
- `Ironman`: Larga Distancia

## Required files

```text
Ironman/outputs/Ironman_Query_Index.json.gz
Ironman/outputs/Ironman_Event_Curves_Index.json.gz
```

`Ironman_Query_Index.json.gz` contains normalized total-time curves, marginal segment curves, segment-pair planes, and swim-bike-run cubes.

`Ironman_Event_Curves_Index.json.gz` contains empirical event-year curves for historical comparisons.
